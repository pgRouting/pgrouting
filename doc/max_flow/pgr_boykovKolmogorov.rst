pgr_boykovKolmogorov
============================================

``pgr_boykovKolmogorov`` â€” Calculates the flow on the graph edges that maximizes the flow from the sources to the targets using Boykov Kolmogorov algorithm.

.. figure:: images/boost-inside.jpeg
   :target: https://www.boost.org/libs/graph/doc/boykov_kolmogorov_max_flow.html

   Boost Graph Inside

.. Rubric:: Availability:

* Version 3.2.0

  * New **proposed** function:

    * pgr_boykovKolmogorov(Combinations)

* Version 3.0.0

  * **Official** function

* Version 2.5.0

  * Renamed from ``pgr_maxFlowBoykovKolmogorov``
  * **Proposed** function

* Version 2.3.0

  * New **Experimental** function
