pgr_version¶

pgr_version — consulta de información sobre la versión de pgRouting.

Disponibilidad

  • Versión 3.0.0

    • Cambio estructural en las columnas de resultados

    • Soporte termina para antigua firma

  • Versión 2.0.0

    • Función oficial

Descripción¶

Devuelve la información de la versión de pgRouting.

Firma¶

pgr_version()
RETURNS TEXT
Ejemplo:

Versión de pgRouting para esta documentación

SELECT pgr_version();
 pgr_version
-------------
 3.5.0
(1 row)

Columnas de Resultados¶

Tipo

Descripción

TEXT

Versión de pgRouting
