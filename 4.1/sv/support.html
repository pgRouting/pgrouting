Support¶

pgRouting community support är tillgängligt via pgRouting webbplats, dokumentation, handledning, e-postlistor och andra. Om du letar efter kommersiellt stöd, hittar du nedan en lista över företag som tillhandahåller pgRouting-utveckling och konsulttjänster.

Rapportering av problem¶

Buggar rapporteras och hanteras i en issue tracker. Vänligen följ dessa steg:

  1. Sök bland ärendena för att se om ditt problem redan har rapporterats. Om så är fallet kan du lägga till extra sammanhang som du kan ha hittat, eller åtminstone ange att du också har problemet. Detta hjälper oss att prioritera vanliga problem.

  2. Om ditt problem inte har rapporterats, skapa en ny fråga <https://github.com/pgRouting/pgrouting/issues/new>`__ för det.

  3. I din rapport inkluderar du uttryckliga instruktioner för att replikera ditt problem. De bästa ärendena innehåller den exakta SQL som krävs för att replikera ett problem.

  4. Om du kan testa äldre versioner av PostGIS för ditt problem, vänligen gör det. Notera den tidigaste versionen som problemet uppträder i ditt ärende.

  5. För de versioner där du kan replikera problemet, notera operativsystemet och versionen av pgRouting, PostGIS och PostgreSQL.

  6. Vi rekommenderar att du använder följande wrapper på problemet för att hitta det steg som orsakar problemet.

SET client_min_messages TO debug;
  <your code>
SET client_min_messages TO notice;

Mailinglista, matris, GIS StackExchange¶

Det finns två e-postlistor för pgRouting som är värd på OSGeo diskurserver:

  • E-postlista för användare: https://discourse.osgeo.org/c/pgrouting/pgrouting-users/

  • E-postlista för utvecklare: https://discourse.osgeo.org/c/pgrouting/pgrouting-dev/

    • Prenumerera: https://discourse.osgeo.org/g/pgrouting-dev

För allmänna frågor och ämnen om hur du använder pgRouting, vänligen skriv till e-postlistan för användare.

Om du vill chatta med andra pgRouting-användare och bidragsgivare kan du gå med i vår matriskanal: pgRouting matriskanal.

Du kan också ställa frågan på GIS StackExchange och märka frågan med pgrouting. Hitta alla frågor taggade med pgrouting under https://gis.stackexchange.com/questions/tagged/pgrouting eller prenumerera på pgRouting questions feed.
