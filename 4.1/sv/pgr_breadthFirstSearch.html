pgr_breadthFirstSearch - Experimentell¶

pgr_breadthFirstSearch - Returnerar traverseringsordning(ar) med algoritmen Breadth First Search.

Experiment

Varning

Experimentella funktioner

  • De är inte officiellt i den aktuella versionen.

  • De kommer troligen inte att officiellt vara en del av nästa utgåva:

    • Funktionerna får inte använda sig av ANY-INTEGER och ANY-NUMERICAL

    • Namnet kan komma att ändras.

    • Signaturen kan komma att ändras.

    • Funktionaliteten kan komma att ändras.

    • pgTap-tester kan saknas.

    • Kan behöva c/c++ granskning.

    • Eventuell dokumentation kan behöva skrivas om.

    • Det kan behövas mycket feedback från samhället.

Tillgänglighet

Version 4.0.0:

  • Standardisering av utdatakolumner till (seq, depth, start_vid, pred, node, edge, cost, agg_cost)

Version 3.0.0

  • Ny experimentell funktion.

Beskrivning¶

Ger genomgångsordningen Breadth First Search från en rotvertex till ett visst djup.

De viktigaste egenskaperna är:

  • Implementeringen kommer att fungera på alla typer av grafer.

  • Ger traverseringsordningen Breadth First Search från en källnod till en måldjupnivå.

  • Speltid: \(O(E + V)\)

Parametrar¶

Parameter

Typ

Beskrivning

Kanter SQL

TEXT

Kanter SQL enligt beskrivningen nedan.

root vid

BIGINT

Identifierare för trädets rotvertex.

  • När värdet är \(0\) får du den spännvidd som börjar i aleatoriska noder för varje träd i skogen.

root vids

ARRAY [ ANY-TAL ]

Array med identifierare för rotpunkterna.

  • \(0\) värden ignoreras

  • För optimeringsändamål ignoreras alla duplicerade värden.

Var:

ANY-TAL:

SMALLINT, INTEGER, BIGINT

VALFRI NUMERISK:

SMALLINT, INTEGER, BIGINT, REAL, FLOAT, NUMERIC
