pgr_full_version¶

pgr_full_version — Obtener los detalles de la información de la versión de pgRouting.

Disponibilidad

  • Versión 3.0.0
    • Nueva función oficial

Soporte

Descripción¶

Obtenga los detalles de la información de la versión de pgRouting

Firmas¶

     pgr_full_version()
RETURNS RECORD OF (version, build_type, compile_date, library, system, PostgreSQL, compiler, boost, hash)
Ejemplo:Información sobre cuándo se construyó esta documentación 
SELECT * FROM pgr_full_version();
 version | build_type | compile_date |     library     |         system          |                 postgresql                  | compiler  | boost  |    hash
---------+------------+--------------+-----------------+-------------------------+---------------------------------------------+-----------+--------+------------
 3.0.6   | Release    | 2021/10/22   | pgrouting-3.0.6 | Linux-5.11.0-38-generic | PostgreSQL 13.4 (Ubuntu 13.4-4.pgdg20.04+1) | GNU-9.3.0 | 1.71.0 | 8bf98ddcc3
(1 row)

Columnas de Resultados¶

Columna Tipo Descripción
version TEXT Versión de pgRouting
build_type TEXT El tipo de Compilación
compile_date TEXT Fecha de compilación
library TEXT Nombre y versión de la biblioteca
system TEXT Sistema operativo
postgreSQL TEXT pgsql utilizado
compiler TEXT Compilador y versión
boost TEXT Versión de Boost
hash TEXT Hash de Git de pgRouting
