pgr_anslutnaKomponenter¶

pgr_connectedComponents - Anslutna komponenter i en odelad graf med hjälp av en DFS-baserad metod.

Tillgänglighet

  • Version 3.0.0

    • Resultatkolumnerna ändras:

      • n_seq tas bort

      • seq har ändrat typ till BIGINT

    • Funktion befordrad till tjänsteman.

  • Version 2.5.0

    • Ny experimentell funktion.

Beskrivning¶

En kopplad komponent i en odelad graf är en uppsättning hörn som alla är nåbara från varandra.

De viktigaste egenskaperna är:

  • Fungerar för undirected grafer.

  • Komponenterna beskrivs av hörn

  • De returnerade värdena är ordnade:

    • komponent stigande

    • nod stigande

  • Speltid: \(O(V + E)\)

Det finns tre grundläggande sätt att ansluta komponenter:

  • Från toppunkten till startpunkten för kanten

  • Från toppunkten till slutpunkten för kanten

  • Från toppunkten till den närmaste toppunkten på kanten

    • Denna lösning kräver att kanten delas.

I detta exempel kommer pgr_separeraKorsning och pgr_separateTouching att användas.

Förbereda tabeller

I det här exemplet behöver tabellen edges en extra kolumn och tabellen vertex byggs om helt och hållet.

ALTER TABLE edges ADD old_id BIGINT;
ALTER TABLE
DROP TABLE vertices;
DROP TABLE

Infoga nya kanter

Använd pgr_separeraKorsning och pgr_separateTouching för att infoga resultaten i tabellen över kanter.

INSERT INTO edges (old_id, geom)
SELECT id, geom FROM pgr_separateCrossing('SELECT * FROM edges')
UNION
SELECT id, geom FROM pgr_separateTouching('SELECT * FROM edges');
INSERT 0 6

Skapa en tabell över hörnpunkter

Skapa tabellen med hjälp av pgr_extractVertices.

CREATE TABLE vertices AS
SELECT * FROM pgr_extractVertices('SELECT id, geom FROM edges');
SELECT 18

Uppdatera topologin

/* -- set the source information */
UPDATE edges AS e
SET source = v.id, x1 = x, y1 = y
FROM vertices AS v
WHERE ST_StartPoint(e.geom) = v.geom;
UPDATE 24
/* -- set the target information */
UPDATE edges AS e
SET target = v.id, x2 = x, y2 = y
FROM vertices AS v
WHERE ST_EndPoint(e.geom) = v.geom;
UPDATE 24

Uppdatera andra värden

I det här exemplet uppdateras endast cost och reverse_cost, där de baseras på geometrins längd och riktningen bibehålls med hjälp av funktionen sign.

UPDATE edges e
  SET cost          = ST_length(e.geom)*sign(e1.cost),
      reverse_cost  = ST_length(e.geom)*sign(e1.reverse_cost)
FROM edges e1
WHERE e.cost IS NULL AND e1.id = e.old_id;
UPDATE 6
