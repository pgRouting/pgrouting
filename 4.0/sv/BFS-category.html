BFS - Kategori¶

  • pgr_kruskalBFS

  • pgr_primBFS

Traversering med hjälp av första breda sökning.

  • Den är endast implementerad på en ej riktad graf.

  • Processen utförs endast på kanter med positiva kostnader.

  • När grafen är ansluten

    • De resulterande kanterna bildar ett träd

  • När grafen inte är ansluten,

    • Hittar ett minsta spännträd för varje ansluten komponent.

    • De kanter som bildas utgör en skog.

Parametrar¶

Parameter

Typ

Beskrivning

Kanter SQL

TEXT

Kanter SQL enligt beskrivningen nedan.

root vid

BIGINT

Identifierare för trädets rotvertex.

  • När värdet är \(0\) får du den spännvidd som börjar i aleatoriska noder för varje träd i skogen.

root vids

ARRAY [ ANY-TAL ]

Array med identifierare för rotpunkterna.

  • \(0\) värden ignoreras

  • För optimeringsändamål ignoreras alla duplicerade värden.

Var:

ANY-TAL:

SMALLINT, INTEGER, BIGINT

VALFRI NUMERISK:

SMALLINT, INTEGER, BIGINT, REAL, FLOAT, NUMERIC
