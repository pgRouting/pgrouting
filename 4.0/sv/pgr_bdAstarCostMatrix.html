pgr_bdAstarCostMatrix¶

pgr_bdAstarCostMatrix - Calculates the cost matrix using pgr_bdAstar.

Tillgänglighet

  • Version 3.0.0

    • Funktion befordrad till tjänsteman.

  • Version 2.5.0

    • Ny föreslagen funktion.

Beskrivning¶

De viktigaste egenskaperna är:

  • Internally uses the pgr_bdAstar algorithm.

  • Returnerar en kostnadsmatris.

  • Ingen beställning utförs

  • låt v och u vara noder i grafen:

    • när det inte finns någon väg från v till u:

      • ingen motsvarande rad returneras

      • kostnaden från v till u är \(\inf\)

    • när \(v = u\) då

      • ingen motsvarande rad returneras

      • kostnaden från v till u är \(0\)

  • När grafen är icke-riktad är kostnadsmatrisen symmetrisk

Boost-graf inuti

Underskrifter¶

pgr_bdAstarCostMatrix(Kanter SQL, start vids, [options])
options: [directed, heuristic, factor, epsilon]
Returnerar uppsättning av (start_vid, end_vid, agg_cost)
ELLER TOM UPPSÄTTNING
Exempel:

Symmetrisk kostnadsmatris för topparna \(\{5, 6, 10, 15\}\) på en omdirigerad graf med heuristiken \(2\)

SELECT * FROM pgr_bdAstarCostMatrix(
  'SELECT id, source, target, cost, reverse_cost, x1, y1, x2, y2 FROM edges',
  (SELECT array_agg(id) FROM vertices WHERE id IN (5, 6, 10, 15)),
  directed => false, heuristic => 2
);

aStar valfria parametrar¶

Parameter

Typ

Standard

Beskrivning

heuristisk

INTEGER

5

Heuristiskt nummer. Aktuella giltiga värden 0~5.

  • 0: \(h(v) = 0\) (Använd detta värde för att jämföra med pgr_dijkstra)

  • 1: \(h(v) = abs(max(\Delta x, \Delta y))\)

  • 2: \(h(v) = abs(min(\Delta x, \Delta y))\)

  • 3: \(h(v) = \Delta x * \Delta x + \Delta y * \Delta y\)

  • 4: \(h(v) = sqrt(\Delta x * \Delta x + \Delta y * \Delta y)\)

  • 5: \(h(v) = abs(\Delta x) + abs(\Delta y)\)

faktor

FLOAT

1

För manipulering av enheter. \(factor > 0\).

epsilon

FLOAT

1

För mindre begränsade resultat. \(epsilon >= 1\).

Se heuristics tillgänglig och factor hantering.
