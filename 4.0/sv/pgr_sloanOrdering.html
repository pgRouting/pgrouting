pgr_sloanOrdering - Experimental¶

pgr_sloanOrdering — Returns the Sloan ordering of an undirected graph

Experimental

Varning

Experimentella funktioner

  • De är inte officiellt i den aktuella versionen.

  • De kommer troligen inte att officiellt vara en del av nästa utgåva:

    • Funktionerna får inte använda sig av ANY-INTEGER och ANY-NUMERICAL

    • Namnet kan komma att ändras.

    • Signaturen kan komma att ändras.

    • Funktionaliteten kan komma att ändras.

    • pgTap-tester kan saknas.

    • Might need c/c++ review.

    • Eventuell dokumentation kan behöva skrivas om.

    • Det kan behövas mycket feedback från samhället.

Tillgänglighet

Version 4.0.0

  • Ny experimentell funktion.

Beskrivning¶

The Sloan ordering algorithm reorders the vertices of a graph to reduce bandwidth, profile, and wavefront properties, which is particularly useful for sparse matrix computations and finite element analysis.

  • Finds a pseudoperipheral vertex pair to determine good starting points

  • Uses a priority-based algorithm that balances vertex degree and distance from the start vertex.

  • Aims to minimize bandwidth (maximum difference between connected vertex indices.

  • Implementationen är för undirected grafer

  • Typically produces better orderings than simple breadth-first approaches.

  • Time complexity: \(O(V² + E)\) where V is the number of vertices and E is the number of edges.
