    • pgr_full_version¶

    pgr_full_version — 获取pgRouting版本信息的详细信息。.

    可用性

    • 版本3.0.0

      • Official function.

    描述¶

    获取 pgRouting 版本信息的完整详细信息

    Boost Graph Inside

    签名¶

    pgr_full_version()
    RETURNS (version, build_type, compile_date, library, system, PostgreSQL, compiler, boost, hash)
    示例:

    有关此文档何时构建的信息

    SELECT version, library FROM pgr_full_version();
  version  |     library
-----------+-----------------
 4.0.0-dev | pgrouting-4.0.0
(1 row)
