    • pgr_breadthFirstSearch - 实验性¶

    pgr_breadthFirstSearch — 使用广度优先搜索算法返回遍历顺序。

    Warning

    描述¶

    提供从根顶点到特定深度的广度优先搜索遍历顺序。

    主要特点是：

    • 该实现适用于任何类型的图。

    • 提供从源节点到目标深度级别的广度优先搜索遍历顺序。

    • 运行时间： \(O(E + V)\)

    多个顶点¶

    参数¶

    参数

    类型

    描述

    Edges SQL

    TEXT

    Edges SQL 如下所述。

    root vid

    BIGINT

    树的根顶点的标识符。

    • 当值为 \(0\) 时，则获取从森林中每棵树的随机节点开始的生成森林。

    root vids

    ARRAY [ ANY-INTEGER ]

    根顶点的标识符数组。

    • \(0\) 值被忽略

    • 出于优化目的，任何重复的值都将被忽略。

    其中：

    ANY-INTEGER:

    SMALLINT, INTEGER, BIGINT

    ANY-NUMERIC:

    SMALLINT, INTEGER, BIGINT, REAL, FLOAT, NUMERIC

    可选参数¶

    列

    类型

    默认

    描述

    directed

    BOOLEAN

    true

    • 当 true 时，该图被视为有 有向

    • 如果为 false ，则该图被视为 无向 。

    DFS 可选参数¶

    参数

    类型

    默认

    描述

    max_depth

    BIGINT

    \(9223372036854775807\)

    树深度的上限。

    • 为负数时抛出错误。

    内部查询¶

    Edges SQL¶

    列

    类型

    默认

    描述

    id

    ANY-INTEGER

    边的标识符。

    source

    ANY-INTEGER

    边的第一个端点顶点的标识符。

    target

    ANY-INTEGER

    边的第二个端点顶点的标识符。

    cost

    ANY-NUMERICAL

    edge (source, target)的权重

    reverse_cost

    ANY-NUMERICAL

    -1

    边(target, source)的权重

    • 当为负时：edge (target, source) 不存在，因此它不是图的一部分。

    其中：

    ANY-INTEGER:

    SMALLINT, INTEGER, BIGINT

    ANY-NUMERICAL:

    SMALLINT, INTEGER, BIGINT, REAL, FLOAT

    结果列¶

    返回一组 (seq, depth, start_vid, node, edge, cost, agg_cost)

    参数

    类型

    描述

    seq

    BIGINT

    从 \(1\) 开始的顺序值。

    depth

    BIGINT

    node 的深度。

    • \(0\) 当 node = start_vid 。

    start_vid

    BIGINT

    根顶点的标识符。

    node

    BIGINT

    使用 edge 到达的``node`` 的标识符。

    edge

    BIGINT

    用于到达 node 的 edge 的标识符。

    • \(-1\) 当 node = start_vid 。

    cost

    FLOAT

    遍历 edge 的成本。

    agg_cost

    FLOAT

    从 start_vid 到 node 的总成本。

    其中：

    ANY-INTEGER:

    SMALLINT, INTEGER, BIGINT

    ANY-NUMERIC:

    SMALLINT, INTEGER, BIGINT, REAL, FLOAT, NUMERIC
