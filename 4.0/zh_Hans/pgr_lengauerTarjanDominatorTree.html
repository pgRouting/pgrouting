    pgr_lengauerTarjanDominatorTree — 返回所有顶点的直接支配者。

    描述¶

    该算法计算每个顶点的 直接支配者**（称为 **idom），一旦计算出每个顶点的 idom，然后通过将每个顶点的 idom 作为其父节点，可以构建支配树。

    主要特点是：

    • 该算法仅适用于有向图。

    • 返回值没有排序。

    • 该算法返回每个顶点的* idom*。

    • 如果图中不存在*根顶点*，则返回空集。

    • 运行时间： \(O((V+E)log(V+E))\)

    总结

    pgr_lengauerTarjanDominatorTree(Edges SQL, root vertex)
    返回 (seq, vertex_id, idom) 的集合
    OR EMPTY SET

    参数¶

    列

    类型

    描述

    Edges SQL

    TEXT

    SQL 查询如上所述。

    root vertex

    BIGINT

    起始顶点的标识符。

    内部查询¶

    Edges SQL¶

    列

    类型

    默认

    描述

    id

    ANY-INTEGER

    边的标识符。

    source

    ANY-INTEGER

    边的第一个端点顶点的标识符。

    target

    ANY-INTEGER

    边的第二个端点顶点的标识符。

    cost

    ANY-NUMERICAL

    边(source, target)的权重

    reverse_cost

    ANY-NUMERICAL

    -1

    边(target, source)的权重

    • 当为负时：边（ target, source ）不存在，因此它不是图的一部分。

    其中：

    ANY-INTEGER:

    SMALLINT, INTEGER, BIGINT

    ANY-NUMERICAL:

    SMALLINT, INTEGER, BIGINT, REAL, FLOAT

    结果列¶

    返回集合 (seq, vertex_id, idom)

    列

    类型

    描述

    seq

    INTEGER

    从 1 开始的顺序值。

    vertex_id

    BIGINT

    顶点的标识符。

    idom

    BIGINT

    顶点的直接支配者。
