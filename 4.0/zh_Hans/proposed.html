    • 提议的功能¶

    Warning

    下一版本的提议功能。

    • 它们并未正式出现在当前版本中。

    • 它们可能会正式成为下一个版本的一部分：

      • 这些函数使用 ANY-INTEGER 和 ANY-NUMERICAL

      • 名字可能不会改变。（但仍然有可能改变）

      • 签名可能不会改变。（但仍然有可能改变）

      • 功能可能不会改变。（但仍然有可能改变）

      • pgTap 测试已经完成。 但可能需要更多。

      • 文档可能需要完善。

    族

    Dijkstra - 函数族

    • pgr_dijkstraVia - 提议中 - 获取经过一系列顶点的路径。

    • pgr_dijkstraNear - 提议中 - 获取到最近顶点的路线。

    • pgr_dijkstraNearCost - 提议中 - 获取最近顶点的成本。

    withPoints - 函数族

    • pgr_withPoints - 从/到图表上任意点的路线。

    • pgr_withPointsCost - 最短路径的成本。

    • pgr_withPointsCostMatrix - 最短路径的成本。

    • pgr_withPointsKSP -K 最短路径。

    • pgr_withPointsDD -行驶距离。

    • pgr_withPointsVia - 通过路由

    转换 - 函数族

    • pgr_lineGraph - 提议中 - 用于生成折线图的转换算法。

    着色 - 函数族

    • pgr_sequentialVertexColoring - 提议中 - 顶点着色算法采用贪心法。

    收缩 - 函数族

    • pgr_contractionDeadEnd - 提议中

    • pgr_contractionLinear - 提议中

    Traversal - 函数族

    • pgr_depthFirstSearch - 提议中 - 图的深度优先搜索遍历。

    类别

    成本 - 类别

    • pgr_withPointsCost

    成本矩阵 - 类别

    • pgr_withPointsCostMatrix

    驾驶距离 - 类别

    • pgr_withPointsDD - 基于 pgr_withPoints 的行驶距离

    K最短路径 - 类别

    • pgr_withPointsKSP - 基于 pgr_withPoints 的 Yen 算法

    Via - 类别

    • pgr_dijkstraVia - 提议中

    • pgr_withPointsVia

    • pgr_trspVia

    • pgr_trspVia_withPoints

    withPoints - 类别

    • withPoints - 函数族 - 基于 Dijkstra 算法的函数。

    • 来自 TRSP - 函数族:

      • pgr_trsp_withPoints - 有限制的顶点/点路由。

      • pgr_trspVia_withPoints - 通过有限制的顶点/点路由。
