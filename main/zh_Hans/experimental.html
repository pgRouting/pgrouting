    Warning

    实验功能

    • 它们不是当前版本的正式版本。

    • 它们可能不会正式成为下一个版本的一部分：

      • 这些函数可能不使用 ANY-INTEGER 和 ANY-NUMERICAL

      • 名称可能会改变。

      • 签名可能会改变。

      • 功能可能会改变。

      • pgTap 测试可能丢失。

      • 可能需要 c/c++编码。

      • 可能缺乏文档。

      • 文档（如果有）可能需要重写。

      • 可能需要自动生成文档示例。

      • Might need a lot of feedback from the community.

      • 可能取决于 pgRouting 的拟议功能

      • 可能依赖于 pgRouting 的已弃用函数

    族

    Flow - 函数族

    • pgr_maxFlowMinCost - 实验 - 边缘上的流量和成本详细信息。

    • pgr_maxFlowMinCost_Cost - 实验 -仅最小成本计算。

    中国邮递员问题 - 函数族（实验）

    • pgr_chinesePostman - 实验

    • pgr_chinesePostmanCost - 实验

    着色 - 函数族

    • pgr_bipartite - Experimental - 使用基于 DFS 的着色方法的二分图算法。

    • pgr_edgeColoring - Experimental - 使用 Vizing 定理的边缘着色算法。

    Transformation - Family of functions

    • pgr_lineGraphFull - 实验 - 用于从输入图中的每个顶点生成线图的变换算法。

    Traversal - 函数族

    • pgr_breadthFirstSearch - 实验 - 图的广度优先搜索遍历。

    • pgr_binaryBreadthFirstSearch - 实验 -图的广度优先搜索遍历。

    分量 - 函数族

    • pgr_makeConnected - 实验 - 使图形连接的边的详细信息。

    排序 - 函数族

    • pgr_cuthillMckeeOrdering`` - 试验性的 -返回无向图的反向 Cuthill-McKee 排序。

    • pgr_topologicalSort - 实验 - Linear ordering of the vertices for directed acyclic graph.

    Metrics - Family of functions

    • pgr_betweennessCentrality - Experimental - Calculates relative betweenness centrality using Brandes Algorithm

    TRSP - 函数族

    • pgr_turnRestrictedPath - Experimental - 有限制的路由。

    类别

    Vehicle Routing Functions - Category

    • 取货和送货问题

      • pgr_pickDeliver - 实验 - 使用成本矩阵的取货&送货

      • pgr_pickDeliverEuclidean - 实验 - 使用欧几里得距离的取货&送货

    • 分配问题

      • pgr_vrpOneDepot - Experimental - 从单个仓库分发订单

    Shortest Path Category

    • pgr_bellmanFord - 实验

    • pgr_dagShortestPath - Experimental

    • pgr_edwardMoore - Experimental

    Planar Family

    • pgr_isPlanar - 实验

    Miscellaneous Algorithms

    • pgr_lengauerTarjanDominatorTree - Experimental

    • pgr_stoerWagner - Experimental

    • pgr_transitiveClosure - 实验

    • pgr_hawickCircuits - Experimental
