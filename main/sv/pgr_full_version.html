pgr_full_version¶

pgr_full_version - Hämta detaljerna i versionsinformationen för pgRouting.

Tillgänglighet

  • Version 3.0.0

    • Officiell funktion.

Beskrivning¶

Få fullständig information om pgRouting version information

Boost-graf inuti

Underskrifter¶

pgr_full_version()
RETURNS (version, build_type, compile_date, library, system, PostgreSQL, compiler, boost, hash)
Exempel:

Information om när denna dokumentation skapades

SELECT version, library FROM pgr_full_version();
   version   |     library
-------------+-----------------
 4.0.0-beta1 | pgrouting-4.0.0
(1 row)
