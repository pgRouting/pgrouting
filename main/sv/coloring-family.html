Färgläggning - Familj av funktioner¶

Föreslagen

Varning

Föreslagna funktioner för nästa version av Mayor.

  • De finns inte officiellt med i den aktuella versionen.

    • Code has been reviewed therefore is not experimental.

    • Name, signature and functionality might not change.

    • pgTap-tester har gjorts. Men det kan behövas fler.

    • Dokumentationen kan behöva förbättras.

  • `pgr_sequentialVertexColoring - Föreslagen - Vertex coloring algorithm using greedy approach.

Experimental

Varning

Experimentella funktioner

  • De är inte officiellt i den aktuella versionen.

  • De kommer troligen inte att officiellt vara en del av nästa utgåva:

    • Funktionerna får inte använda sig av ANY-INTEGER och ANY-NUMERICAL

    • Namnet kan komma att ändras.

    • Signaturen kan komma att ändras.

    • Funktionaliteten kan komma att ändras.

    • pgTap-tester kan saknas.

    • Might need c/c++ review.

    • Eventuell dokumentation kan behöva skrivas om.

    • Det kan behövas mycket feedback från samhället.

  • pgr_bipartite - Experimentell - Bipartite graph algorithm using a DFS-based coloring approach.

  • pgr_edgeColoring - Experimentell - Edge Coloring algorithm using Vizing’s theorem.
