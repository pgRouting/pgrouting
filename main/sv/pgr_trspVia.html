pgr_trspVia¶

pgr_trspVia Rutt som går igenom en lista med toppar med restriktioner.

Tillgänglighet

Version 4.0.0

  • Funktion befordrad till tjänsteman.

  • Förändring på gång, signaturer inte längre tillgängliga:

    • pgr_trspviavertices(text,anyarray,boolean,boolean,text)

Version 3.4.0

  • Ny föreslagen funktion.

  • Föråldrade signaturer

    • pgr_trspviavertices(text,anyarray,boolean,boolean,text)

Beskrivning¶

Givet en lista med toppar och en graf är denna funktion likvärdig med att hitta den kortaste vägen mellan \(vertex_i\) och \(vertex_{i+1}\) för alla \(i < size\_of(via\;toppar)\) och försöker att inte använda begränsade vägar.

Banorna representerar de olika delarna av rutten.

Den allmänna algoritmen är som följer:

  • Exekvera en pgr_dijkstraVia.

  • För uppsättningen delvägar för lösningen som passerar genom en begränsning gäller följande

    • Utför TRSP-algoritmen med begränsningar för banorna.

    • NOTE när detta görs ignoreras flaggan U_turn_on_edge.

Boost-graf inuti

Underskrifter¶

En Via¶

pgr_trspVia(Edges SQL, Restrictions SQL, via vertices, [options])
options: [directed, strict, U_turn_on_edge]
Returnerar uppsättning av (seq, path_id, path_seq, start_vid, end_vid, node, edge, cost, agg_cost, route_agg_cost)
ELLER TOM UPPSÄTTNING

Från pgr_dijkstraVia-resultatet tar den bort de motstridiga vägarna och bygger lösningen med resultaten från pgr_trsp-algoritmen:

WITH
solutions AS (
  SELECT path_id, path_seq, start_vid, end_vid, node, edge, cost, agg_cost FROM pgr_dijkstraVia(
    $$SELECT id, source, target, cost, reverse_cost FROM edges$$,
    ARRAY[6, 3, 6]) WHERE path_id != 1
  UNION
  SELECT 1 AS path_id, path_seq, start_vid, end_vid, node, edge, cost, agg_cost FROM pgr_trsp(
    $$SELECT id, source, target, cost, reverse_cost FROM edges$$,
    $$SELECT path, cost FROM restrictions$$,
    6, 3)),
with_seq AS (
  SELECT row_number() over(ORDER BY path_id, path_seq) AS seq, *
  FROM solutions),
aggregation AS (SELECT seq, SUM(cost) OVER(ORDER BY seq) AS route_agg_cost FROM with_seq)
SELECT with_seq.*, COALESCE(route_agg_cost, 0) AS route_agg_cost
FROM with_seq LEFT JOIN aggregation ON (with_seq.seq = aggregation.seq + 1);
