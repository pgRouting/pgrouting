pgr_version¶

Nombre¶

pgr_version — consulta de información sobre la versión de pgRouting.

Sinopsis¶

Devuelve una tabla con la información sobre la versión de pgRouting.

table() pgr_version();

Descripción¶

Devuelve una tabla con:

version:varchar versión de pgRouting
tag:varchar La etiqueta de la construcción en Git de pgRouting
hash:varchar El Hash de la construcción en Git de pgRouting
branch:varchar La ramificación de la construcción en Git de pgRouting
boost:varchar versión de Boost

Historia

  • Nuevo en la versión 2.0.0

Ejemplos¶

  • Consulta para la cadena con la información completa de la versión.
SELECT pgr_version();

                     pgr_version
------------------------------------------------------
 (2.0.0-dev,v2.0dev,290,c64bcb9,sew-devel-2_0,1.49.0)
(1 row)
  • Consulta para los atributos version y boost
SELECT version, boost FROM pgr_version();

  version  | boost
-----------+--------
 2.0.0-dev | 1.49.0
(1 row)
